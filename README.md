Aplicação de xadrez desenvolvida em **Java puro**, executada via **prompt de comando**, com foco em **Programação Orientada a Objetos, modelagem de domínio e separação de responsabilidades**.

O sistema permite a execução de **uma partida completa de xadrez**, respeitando todas as regras oficiais do jogo, incluindo movimentos especiais e validações.

## Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento, herança e polimorfismo
- Separação clara de responsabilidades
- Modelagem de domínio
- Lógica de negócio independente de frameworks
- Tratamento de exceções para regras inválidas

## Funcionalidades

- Execução de partida completa de xadrez via terminal
- Movimentação válida de todas as peças
- Validação de movimentos inválidos
- Tratamento de exceções para jogadas incorretas
- Movimentos especiais do xadrez, incluindo:
  - Promoção de peão ao alcançar o outro lado do tabuleiro
  - (adicione aqui roque / en passant / xeque-mate se existirem)
- Controle de turnos e estado da partida

## Estrutura do projeto

O projeto é organizado em camadas bem definidas, separando:
- entidades do domínio do xadrez
- regras do jogo
- lógica de execução
- interação com o usuário via prompt de comando

Essa estrutura facilita a leitura, manutenção e evolução do sistema.
